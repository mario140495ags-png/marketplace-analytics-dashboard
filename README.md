# marketplace-analytics-dashboard
Marketplace metrics dashboard using PostgreSQL (Neon) + Grafana. Personal project to practice SQL, visualization, and data modeling.

1. Weekly Performance Ovrview
The graphic shows a week-by-week performance overview of user behavior and platform activity. It tracks five key metrics simultaneously:
-- Users in cohort (green)
-- Users with orders (yellow)
-- Conversion rate (blue)
-- Average sessions (orange)
-- Average ticket final (red)
